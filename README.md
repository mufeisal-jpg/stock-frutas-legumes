# Stock — Frutas & Legumes

App simples para apontar stock diário (quantidade em campo livre).

## Como correr localmente
1) Instalar Node.js (LTS)
2) `npm install`
3) `npm run dev`

## Publicar na Vercel (via GitHub)
- Subir esta pasta para um repositório no GitHub
- Na Vercel: Add New -> Project -> Import Git Repository -> Deploy
